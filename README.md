### Details
This is a **`simple note-taking application`** which **`only supports text input`** at the moment. The user **`must be logged in`** in order to use this app.

##### Actions implemented (v-0.0.1):
  - Login
  - Sing Up
  - Logout
  - Create Note
  - Delete Note
  - Save Note
  - Serch Note
